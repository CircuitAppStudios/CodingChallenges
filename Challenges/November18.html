November 2018 - "CPU Mania"

Welcome to the November 2018 Coding Challenge. This one is all about a CPU you need to make in Software!


Offline / Printable Info available on November 1st, 2018 when coding begins.


Key Dates:

October 31st, 2018 - Preliminary Info Released (Research Day)
November 1st, 2018 - Coding Begins
November 5th, 2018 - Trials Given
November 15th, 2018 - Submissions Open
November 30th, 2018 - Submissions Close
December 3rd, 2018 - Preliminary Results
December 7th, 2018 - Final Results


Rules:

  1. No Copying
  2. No Teaming
  3. No Cheating
  4. No Hard Coding Results
  5. No Late Submissions
  6. Have Fun!


Requirements (Failing to do so may cause disqualification):

  • Written in C / C++ / Java
  • CMake or Makefile (Only for C or C++)
  • GCC Compatible (Only for C or C++)
  • Must be finialized by November 31st, 2018 - No Editing After


Preliminary Info:

This challenge requires you to make a CPU in software that follows a specification given on November 1st, 2018 and can run the over 50 Trials given on November 5th, 2018. This will act like an emulator in many aspects.
